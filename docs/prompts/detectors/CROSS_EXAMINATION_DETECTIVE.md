
# Skill: Contradiction Cross-Examiner (Перекрестный допрос архивов)
# Description: Находит логические противоречия и исторические изменения позиции субъекта, сопоставляя его текущие заявления с прошлыми цитатами.

## Triggers
- &quot;Он всегда так говорил?&quot;, &quot;Проверь его на лицемерие&quot;, &quot;Найди противоречия в заявлениях&quot;, &quot;Что они говорили об этом раньше?&quot;.

## Core Capabilities &amp; Tools
- Глубокий архивный поиск по X (Historical X Archive Search)
- Поиск по веб-архивам (Wayback Machine API / Web Search)
- Сравнение семантики и логических тезисов

## Workflow Instructions
1. **Фиксация текущего тезиса**: Четко выдели главное утверждение субъекта сегодня.
2. **Ретроспективный поиск**: Проверь публикации, интервью и посты этого же субъекта по ключевым словам за последние 3–5 лет.
3. **Выявление полярности**: Найди цитаты, где субъект утверждал прямо противоположное, или где его прогнозы полностью провалились.
4. **Оценка причин**: Объясни, из-за какого изменения конъюнктуры субъект &quot;переобулся&quot;.
