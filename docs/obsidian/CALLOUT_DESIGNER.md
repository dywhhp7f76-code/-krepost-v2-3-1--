# Skill: Дизайнер блоков Callout (Obsidian Callout Stylist)
# Description: Верстает заметки с использованием стандартных и кастомных CSS-блоков Callout с иконками и сворачиванием.

## Triggers
- &quot;Сделай красивый блок текста&quot;, &quot;Код для Callout в Obsidian&quot;, &quot;Сворачиваемый блок&quot;, &quot;Подсветка цитаты&quot;.

## Workflow Instructions
1. Выбери подходящий тип блока по смыслу: `[!info]`, `[!warning]`, `[!danger]`, `[!note]`, `[!success]`, `[!tip]`.
2. Добавь знак плюс `+` или минус `-` сразу после типа блока, чтобы сделать его сворачиваемым (например, `&gt; [!info]- Скрытый текст`).
3. Сверстай внутренности блока (списки, цитаты, код).

## Response Template
### 🎨 Верстка красивой панели
```markdown
&gt; [!tip]+ 💡 Важное замечание по интеграции
&gt; Внутри этого блока можно использовать **жирный текст** или списки:
&gt; - Пункт А
&gt; - Пункт Б
&gt; &gt; [!warning]- Вложенное предупреждение
&gt; &gt; Будьте осторожны с синтаксисом.
```
