# Вопросы по реструктуризации

## 1. Git mv через API
**Проблема:** GitHub API не поддерживает `git mv` напрямую. Можно только создавать/удалять файлы.
**Вопрос:** Как сохранить историю файлов при перемещении через API?
- Вариант A: Создавать копии в новых местах + удалять старые (теряется история в GitHub UI)
- Вариант B: Использовать git команды напрямую (не доступно через текущий API)
- Вариант C: Принять потерю истории для API-операций

**Решение:** Пока использую Вариант A с явным указанием в коммитах, что это перемещение.

**Решение:** Пока перемещаю в `research/` без шифрования, жду уточнения.
